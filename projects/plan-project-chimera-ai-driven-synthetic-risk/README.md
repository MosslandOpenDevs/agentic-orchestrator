## Getting Started

### Installation

1.  Clone the repository:

    ```bash
    git clone [repository URL]
    cd plan-project-chimera-ai-driven-synthetic-risk
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

### Setup

1.  Set up your PostgreSQL database.  Create a database named `chimera_db` and ensure you have the necessary credentials.
2.  Configure environment variables: Create a `.env` file in the root directory and add your database connection string and any other required API keys.
