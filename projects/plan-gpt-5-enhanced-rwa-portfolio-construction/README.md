## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone [repository URL]
   cd plan-gpt-5-enhanced-rwa-portfolio-construction
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Setup

1.  **Database Setup:**  Set up a PostgreSQL database.  You'll need to configure the database connection details in the `.env` file.
2.  **Environment Variables:** Create a `.env` file in the root directory and set the following variables:
    *   `DATABASE_URL`: Your PostgreSQL connection string.
    *   `ETH_RPC_URL`: Your Ethereum RPC URL (e.g., Infura, Alchemy).
    *   `GPT_API_KEY`: Your OpenAI GPT-5 API key (if applicable).
3.  **Run the Development Server:**
    ```bash
    npm start
    ```
