## Getting Started

### Installation

1.  Clone the repository: `git clone [repository URL]`
2.  Navigate to the project directory: `cd plan-idea_title-claude-powered-mossland-nft`

### Setup

1.  **Install Dependencies:** `npm install` or `yarn install`
2.  **Set up Environment Variables:** Configure the following environment variables:
    *   `ETH_RPC_URL`:  The URL of your Ethereum RPC provider (e.g., Infura, Alchemy).
    *   `CONTRACT_ADDRESS`:  The address of the Mossland NFT contract.
    *   `PRIVATE_KEY`:  Your Ethereum private key (use with extreme caution!).
3.  **Database Setup:**  Ensure you have PostgreSQL installed and running.  Create a database for the application.
