## Getting Started

### Installation

1.  Clone the repository:

    ```bash
    git clone [repository_url]
    cd plan-real-time-stablecoin-authenticity
    ```

2.  Install dependencies:

    ```bash
    npm install
    ```

### Setup

1.  Set up your PostgreSQL database.
2.  Install Node.js and npm.
3.  Configure environment variables (e.g., database URL, API keys).  See `.env.example` for guidance.

## Usage Examples

*   **Running the Development Server:**

    ```bash
    npm run dev
    ```

    This will start the React frontend and the FastAPI backend.
