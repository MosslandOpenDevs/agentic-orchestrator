## Getting Started

### Installation

1.  Clone the repository: `git clone [repository_url]`
2.  Navigate to the project directory: `cd plan-mossland-s-strategic-advantage`

### Setup

1.  **Prerequisites:** Ensure you have Node.js, npm/yarn, Python, and PostgreSQL installed.
2.  **Database Setup:**  Create a PostgreSQL database named `mossland_db` (or adjust the connection string in `config.py`).
3.  **Install Dependencies:** `npm install` or `yarn install`
4.  **Backend Setup:**
    *   `cd backend`
    *   `pip install -r requirements.txt`
    *   Run the FastAPI application: `uvicorn main:app --reload`
5.  **Frontend Setup:**
    *   `cd frontend`
    *   `npm start` or `npm run dev` (depending on your chosen development server)
